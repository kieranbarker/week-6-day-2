# Week 6 Day 2

Clone this repository:

```sh
git clone https://github.com/kieranbarker/week-6-day-2.git
cd week-6-day-2
```

Install the dependencies:

```sh
npm install
```

Seed the database:

```sh
npm run seed
```

Start the server:

```sh
npm run dev
```

Press <kbd>CTRL</kbd> + <kbd>C</kbd> to stop the server when you're done.
